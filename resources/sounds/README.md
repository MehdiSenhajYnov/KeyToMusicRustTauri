# Resources Sounds

Ce dossier doit contenir :

- **error.mp3** : Son d'erreur système joué quand un fichier audio n'est pas trouvé ou qu'une erreur se produit

Le fichier error.mp3 doit être :
- Format : MP3
- Durée : < 1 seconde
- Volume : Modéré (pas trop fort)
- Taille : < 100 KB

Ce fichier sera copié vers les ressources de l'application pendant le build.
